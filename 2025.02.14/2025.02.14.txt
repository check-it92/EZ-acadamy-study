- 일반객체 & 생성자 함수 & Class

new Array()는 자바스크립트에서 내장된 Array 생성자 함수를 사용하여 새로운 배열 객체를 만들기 때문에 배열 형식이 됩니다.  new Array() = 프로토타입 객체로 생성된다.

자세히 설명하면:

Array 생성자 함수: 자바스크립트는 Array라는 내장 생성자 함수를 제공하며, 이 함수를 new 키워드와 함께 호출하면 새로운 배열 인스턴스가 생성됩니다.

특별한 객체: 자바스크립트에서 배열은 기본적으로 객체(object)입니다. 하지만 배열은 인덱스 기반의 요소 접근, length 프로퍼티, push, pop 등의 메서드를 제공하는 특별한 객체입니다.

배열 리터럴: 보통 let arr = [];와 같이 배열 리터럴 표기법을 많이 사용하지만, new Array()도 동일하게 배열을 생성하는 방법입니다.

따라서 let arr = new Array()는 배열 객체를 생성하여 배열 형식으로 사용할 수 있게 해줍니다.

>> 우리는 그래서 arr 이라는 함수를 인스턴스 함수라 불리운다.
