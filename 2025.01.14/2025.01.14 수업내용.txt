 위치 속성을 적용하고 싶은 대상의 부모요소에게 항상 & 반드시 display : flex 부여해줘야한다.
 display : flex 라는 위치속성을 부여하는 순간, 자동으로 중심축 & 교차축이 생성!!
	    부모 밑에 모든 자식요소들이 가로로 한줄 정렬이 됩니다. (그럼,<구 기능> float left,right를 안써도 된다.)

중심축 & 교차축 :
- 중심축 : 가로 row
- 교차축 : 세로 column

**중심축을 기준으로 무언갈 하겠다.  
justify-content: center	(가운데 정렬)	
종류들 : flex-start(좌측정렬). flex-end(우측정렬)
	    flex-direction : (row, column) 정렬의 방향을 가로로 // 세로로 할지 결정지어 주는 요소

	    space-between ( )   /  space-around (자식공간의 둘레가 동일하게 들어간다.)- 일정한간격으로 쪼갤때 (좁다)
	    space-evenly (기준으로 부터 딱 떨어지는 공간만큼) - 좌우측만 떨어뜨려도 될 경우 (넓다)

**교차축을 기준으로 무언갈 하겠다. 
align-items: center(가운데 정렬)
종류들 : flex-start(좌측정렬). flex-end(우측정렬)
	   stretch(길게 늘일때) - 기본값으로 부모의 속성을 이어받고 높이값 입력x 반응형 일때 
	   space-between ( )   /  space-around  / space-evenly

align-self: 이건 부모요소랑 별개로 자식요소 독립적으로 쓰고자 할때 쓰인다.

교차 되는 기준을 꼭 입력을 해야 한다.
세로 값(height) 높이를 부여해줘야 그 기준으로 정렬이 들아간다.
가로는 생략가능하나 형식상 써줄수 있다.

>** 레이아웃 정렬
 을 해줄때 하나의 대상이 묶여있다면(박스구조) 정렬이 쉬우나 여러개면 각각의 값을 적용해야해서
div 값으로 묶어서 사용하는 게 좋다.

>>flex-wrap: wrap <-> no wrap (보장안해준다.) = default 기본값
자식요소의 고정 (너비) 가로 값을 보장해주는 요소
(랩으로 감싼)

flex-flow : flex-direction, flex-wrap 를 한번에 축약해주는 문장
예시) flex-flow: row wrap;  // 이런 식으로 상기 두가지의 요소를 한번에 사용가능 (빈도수 적음)

flex-wrap: wrap 으로 묶었을때 자식요소들의 개별 요소들의 변수는
** align-content vs items
 컨텐츠는 하나의 그룹으로 판단이 되고 // 아이템즈는 각각의 요소로 판단이 된다.

** 제일 헷갈리는 3대장
-자식요소가 부모요소를 뚫고 설정하는 경우는 없다. (극히 드문데 개중에는 본인 실수가 크다.)

flex-grow : 부모요소가 가지고 있는 전체 너비값을 기준으로 그밑에 있는 자식요소들의 너비를 
		몇 분의 몇으로 (=n/1)로 정의 해서 적용하고 싶은 경우!! // 증가하는 비율
				전체 비율을 몇대 몇으로 나눠서 쓸것이냐?

flex-shrink : 부모요소가 가지고 있는 전체 너비값을 기준으로 그 밑에 있는 자식요소들의
		너비의 줄임, 좁아지는 비율을 적용하고 싶은 경우 // 감소하는 비율

flex-basis : 부모요소 아래 있는 자식요소들의 기본 너비값을 어떻게 보장하고 할지 결정 할때!!
 		(=width 의 값과 동일하다)

축약형 : flex : 글로우 슈링크 베이시스 순으로 작성 //  예) flex: 1 1 100px 순으로 작성
