1. 기존 React-router-dom 에서 페이지 라우팅 하는 방법

- 최상위 루트 컴포넌트에 BrowserRouter을 생성 & 하위 컴포텉느 요소들을 모두 감싸안도록 해줌

-  각 페이지별 공통 요소를 제외한 나머지 페이지 라우트 컴포넌트들을 설정하는 방법

> Routes > Route를 활용해서 path&element 속성사용 => path경로별 컴포넌트 페이지 정의!!

2. 새롭게 변경된 React-router-dom에서 페이지 라우팅 하는 방법

- 최상위 루트폴더 > createBrowserRouter을 사용해서 동적경로가 존재하지 않는 메인 페이지
  라우팅을 설정 

- createBrowserRouter는 BrowserRouter 처럼 직접 자식컴포넌트를 감싸안을수 없고, 내부 인자
  값으로 정의한 사항을 별도의 변수에 적용해서 컴포넌트들의 페이지 라우팅을 정의 할 수 있음!

- 설정한 이후에 자식요소의 컴포넌트들에게 children이라는 속성을 활용해서 동적경로별 페이지를
  분기하는 방법!

- 자식요소의 컴포넌트들을 적용하려면, 별도의 Outlet컴포넌트를 사용해야 한다.
